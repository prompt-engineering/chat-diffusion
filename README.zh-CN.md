## 目前仅支持客户端（浏览器）访问 OpenAI，服务器端调用正在开发中

## 本地搭建

1. 从 GitHub 克隆 [ChatVisualNovel](https://github.com/prompt-engineering/chat-diffusion)。
2. 执行 `npm install`。
3. 直接运行 `npm run dev` 就可以使用了。
